# ReSTCV Project Page

This folder is a standalone GitHub Pages project page for:

**Spatio-Temporal Control Variates with ReSTIR for Real-Time Rendering**

## Local Preview

Open `index.html` directly in a browser, or run a tiny local server:

```powershell
cd project-page
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages Deployment

Recommended options:

1. Copy the contents of `project-page/` to the root of a GitHub Pages repository.
2. Or keep this folder in the repo and configure GitHub Pages to publish from this directory if your setup supports it.

The page is static and has no build step.
